## About

👋 Fullstack software engineer focused on the web. I build applications end-to-end, from database schema to rendered pixel.

Currently a Software Engineer at Ensolvers, shipping fullstack features across Spring Boot APIs and a React frontend, and building AI integrations like Retrieval-Augmented Generation for LLM grounding. I lean on solid fundamentals — clean code, design patterns, careful database modeling — and prefer to fully understand a problem before reaching for a fix.

Open to interesting collaborations and conversations on fullstack architecture, RAG, or design systems. If you're building something in that space, feel free to reach out.
